# PalateAI

PalateAI is an AI-powered personal food memory assistant that helps users log dishes, extract menu items from receipts, and learn their taste preferences over time.

## Vision

PalateAI evolves from a dish log into a predictive taste intelligence engine that provides recommendations, price trends, and personalized insights.

## MVP

- Google OAuth authentication
- Receipt/menu photo upload
- AI-based extraction of dish names and prices
- Editable review workflow
- Personal dish history and trend tracking
